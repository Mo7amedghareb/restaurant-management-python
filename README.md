# Restaurant Management System

A console-based restaurant management system built with Python to demonstrate Object-Oriented Programming (OOP) concepts. The application simulates a simple restaurant workflow through a command-line interface.

## Features

- Menu Management
- Order Processing
- Customer Interaction
- Object-Oriented Design
- Console-Based User Interface

## Technologies

- Python
- Object-Oriented Programming (OOP)

## OOP Concepts Applied

- Classes & Objects
- Encapsulation
- Inheritance *(if used)*
- Polymorphism *(if used)*
- Methods & Constructors

## Purpose

This project was developed to practice Python programming and apply object-oriented programming principles by building a simple restaurant management system.
